# slide-style-rector

スライドスタイルガイドに基づいてMarpスライドのレイアウトを美しく整形するスキル。

## 概要

`slides/example.md` の40種類のレイアウトパターンから最適なものを選んで適用する。
**独自レイアウトを作成せず、example.mdのパターンを使うことでデザインの一貫性を維持する。**

## 参照ファイル

| ファイル | 内容 |
|---------|------|
| `slides/example.md` | 40種類のレイアウトパターン（実装例） |
| `docs/style-guide.md` | カラー、タイポグラフィの詳細 |

## 使用手順

1. まず `slides/example.md` を読み込む
2. スライドの内容に最適なパターンを選択する
3. パターンのHTMLをコピーして内容を置き換える
4. `docs/style-guide.md` のカラー・フォント規定に従う

## パターンカテゴリ（40種）

| カテゴリ | パターン数 | 用途 |
|---------|-----------|------|
| **A. タイトル・セクション系** | 5種 | 表紙、章区切り、まとめ、目次、クロージング |
| **B. カラムレイアウト系** | 8種 | 2〜5カラム、グリッド配置 |
| **C. 縦並びリスト系** | 4種 | ステップ、タイムライン、アイコン付き |
| **D. パネルデザイン系** | 5種 | 基本、強調、ガラス風、引用、カード |
| **E. 背景・画像系** | 4種 | 全画面、左右分割、QRコード、オーバーレイ |
| **F. 強調・特殊系** | 3種 | 数値統計、中央配置、Q&A |
| **G. 応用パターン** | 10種 | フロー図、対比、企業事例、チェックリストなど |

## デザイン原則

### カラーパレット

```
アクセントカラー:
- 濃紺: #1B4565（タイトル背景、見出し）
- ティール: #3E9BA4（アクセント、強調）
- グラデーション: linear-gradient(to right, #1B4565, #3E9BA4)

グレースケール:
- gray-50: パネル背景（メイン）
- gray-100: パネル背景（やや濃い）
- gray-600: サブテキスト
- gray-800: メインテキスト
```

### フォントサイズ（text-em-*クラス）

| クラス | 用途 |
|--------|------|
| `text-em-3xl` | 数値強調、インパクトのあるキーワード |
| `text-em-2xl` | パネル見出し、セクションタイトル |
| `text-em-xl` | サブ見出し |
| `text-em-lg` | 本文、リスト項目 |
| `text-em-base` | 補足説明 |

### パネルスタイル

```html
<!-- 標準パネル -->
<div class="bg-gray-50 rounded-xl shadow p-6">

<!-- 強調パネル（グラデーション） -->
<div class="rounded-xl shadow-lg p-6"
     style="background: linear-gradient(135deg, #1B4565 0%, #3E9BA4 100%);">

<!-- 左ボーダー付きパネル -->
<div class="bg-gray-50 rounded-xl shadow p-6 border-l-4 border-gray-400">
```

### グラデーション背景スライド

```markdown
<!--
_backgroundImage: "linear-gradient(to right, #1B4565, #3E9BA4)"
_color: #fff
-->
```

## 禁止事項

- 独自のレイアウトを新規作成しない
- example.mdにないカラーを使用しない
- 派手な色（赤、オレンジ、黄）を使用しない
- 1スライドあたりアクセントカラー3色以上を使用しない

## パターン選択ガイド

| 内容 | 推奨パターン |
|------|-------------|
| 表紙 | A-1. タイトルスライド |
| 章の切り替え | A-2. グラデーション背景セクション |
| 2つの対比 | B-6. 2カラム比較 or B-7. 2カラム対比 |
| 3つの概念 | B-8. 3カラムレイアウト or B-9. 3カラム（アクセントカラー） |
| 手順説明 | C-14. ステップ表示 |
| 経歴・沿革 | C-15. タイムライン |
| 統計データ | F-27. 数値統計表示 |
| 重要メッセージ | D-19. 強調パネル |
| 名言引用 | D-21. 引用パネル |
| 企業紹介 | G-34. 企業事例 |
| 最終まとめ | G-32. まとめスライド（番号付き） |
| 質疑応答 | F-29. Q&Aスライド |
| 終了 | A-5. クロージングスライド |

## 実行例

```
ユーザー: このスライドを整形して
→ example.mdを読み込み、内容に合うパターンを選択して適用
```
