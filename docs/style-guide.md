# スライドデザイン スタイルガイド

サンプルスライド「AIエージェント時代を生き抜くエンジニアのサバイバル戦略」の分析に基づくデザインガイドラインです。

---

## デザイン哲学

- **美麗でシンプル**：情報過多を避け、視覚的にクリーン
- **色数制限**：グレースケール基調に限定的なアクセントカラー
- **可読性重視**：フォントサイズと余白を適切に
- **一貫性**：パターンの繰り返しで理解しやすく

---

## 1. カラーパレット

### 使用ルール

- **1スライドあたりアクセントカラーは1-2色まで**
- グレースケールを基調とする
- 派手な色（赤、オレンジ、黄）は避ける

### アクセントカラー

| 名前 | 色 | Hex | 用途 |
|------|-----|-----|------|
| **濃紺** | ダークネイビー | `#1B4565` | タイトル背景、見出し |
| **ティール** | ティール | `#3E9BA4` | アクセント、強調 |
| **グラデーション** | 濃紺→ティール | `#1B4565` → `#3E9BA4` | セクション背景 |

```css
/* プライマリグラデーション */
background: linear-gradient(to right, #1B4565, #3E9BA4);
```

### グレースケール

| Tailwindクラス | 用途 | 説明 |
|----------------|------|------|
| `gray-50` | パネル背景 | 淡いグレー（メイン） |
| `gray-100` | パネル背景 | やや濃いグレー |
| `gray-200` | ボーダー | 区切り線 |
| `gray-400` | アクセント | 軽い強調 |
| `gray-500` | 補足テキスト | キャプション、注釈 |
| `gray-600` | サブテキスト | 副次的な本文 |
| `gray-700` | 見出しテキスト | 小見出し |
| `gray-800` | メインテキスト | 本文、タイトル |

## 2. フォントサイズ

| クラス | 用途 |
|--------|------|
| `text-em-3xl` | 数値強調、インパクトのあるキーワード |
| `text-em-2xl` | パネル見出し、セクションタイトル |
| `text-em-xl` | サブ見出し |
| `text-em-lg` | 本文、リスト項目 |
| `text-em-base` | 補足説明 |

## 3. レイアウトパターン

### 基本構成

| パターン | 使用頻度 | 用途 |
|----------|----------|------|
| **1カラム** | 高 | タイトル、セクション区切り、引用 |
| **2カラム** | 最高 | 比較、Before/After、左右分割 |
| **3カラム** | 中 | 3つの概念・ステップの並列表示 |
| **4カラム** | 低 | 4項目の比較（罠4つなど） |

### 典型的なレイアウト

#### タイトルスライド
```
┌─────────────────────────────────┐
│                                 │
│     [大きなタイトル]             │
│     [サブタイトル]               │
│                                 │
│              [著者名・所属]      │
└─────────────────────────────────┘
```

#### コンテンツスライド（2カラム）
```
┌─────────────────────────────────┐
│ [見出し]                        │
│ [サブ見出し]                    │
├───────────────┬─────────────────┤
│   パネル1     │    パネル2      │
│   ・項目      │    ・項目       │
│   ・項目      │    ・項目       │
└───────────────┴─────────────────┘
```

#### まとめスライド
```
┌─────────────────────────────────┐
│ [見出し]                        │
├─────────────────────────────────┤
│ ❶ ポイント1                     │
│    [補足テキスト]               │
├─────────────────────────────────┤
│ ❷ ポイント2                     │
│    [補足テキスト]               │
└─────────────────────────────────┘
```

## 4. パネルデザイン

### カード/ボックススタイル

| 要素 | 値 | 備考 |
|------|-----|------|
| **角丸** | 8-12px | 柔らかい印象 |
| **影** | なし or 非常に薄い | フラットデザイン基調 |
| **ボーダー** | なし or 左辺のみ | 左辺にアクセントカラーのボーダー（4px） |
| **背景色** | 薄いグレー `#F5F7FA` | 白背景との差別化 |
| **内側余白** | 20-24px | 十分な余白を確保 |

### パネルの種類

#### 1. 標準パネル（グレー背景）
```css
background: #F5F7FA;
border-radius: 8px;
padding: 20px;
```

#### 2. アクセントパネル（左ボーダー付き）
```css
background: #F5F7FA;
border-left: 4px solid #3D8B8B;
border-radius: 0 8px 8px 0;
padding: 20px;
```

#### 3. 強調パネル（色付き背景）
```css
background: linear-gradient(135deg, #3D8B8B, #4AA0A0);
color: white;
border-radius: 8px;
padding: 24px;
```

#### 4. 引用パネル
```css
background: white;
border-left: 4px solid #2D7DD2;
padding: 16px 20px;
font-style: italic;
```

## 5. その他のデザイン要素

### アイコン・図形
- 円形のアイコンコンテナ（淡いピンク/ブルー背景）
- 矢印は太めで視認性を確保
- グラフは棒グラフ・円グラフをシンプルに

### 画像の扱い
- 2カラムレイアウトで左テキスト・右画像
- 背景画像使用時は半透明オーバーレイで可読性確保

### ページ番号
- 右下に配置
- フォントサイズ: 14px
- 色: グレー or 背景に応じて白

### 余白
- スライド端からの余白: 40-60px
- 要素間の余白: 20-30px
- パネル内の余白: 20-24px

## 6. Tailwind CSS クラス例

```html
<!-- 標準パネル -->
<div class="bg-gray-50 rounded-lg p-5">

<!-- アクセントパネル（左ボーダー） -->
<div class="bg-gray-50 border-l-4 border-teal-600 rounded-r-lg p-5">

<!-- 2カラムレイアウト -->
<div class="grid grid-cols-2 gap-6">

<!-- 3カラムレイアウト -->
<div class="grid grid-cols-3 gap-4">

<!-- 見出し -->
<h1 class="text-3xl font-bold text-slate-800">

<!-- サブ見出し -->
<h2 class="text-xl font-semibold text-teal-700">

<!-- 本文 -->
<p class="text-base text-gray-700">

<!-- 強調テキスト -->
<span class="font-bold text-slate-900">
```

## 8. スライドタイプ別テンプレート

### タイトルスライド
- 背景: グラデーション（ティール系）
- タイトル: 白、48-56px、太字
- サブタイトル: 白、24px
- 著者: 右下寄せ、白、18px

### セクション区切り
- 背景: グラデーション
- タイトル: 白、40-48px、太字
- サブテキスト: 白、18-20px

### コンテンツスライド
- 背景: 白
- 見出し: ネイビー、32-40px
- パネル: グレー背景、角丸8px

### まとめスライド
- 背景: 淡いグラデーション or 白
- 番号付きリスト（❶❷❸）
- 各項目をパネルで囲む
